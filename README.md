# 하스스톤 카드 퀴즈
+ /하스카드퀴즈 명령어를 통해 퀴즈를 실행할 수 있습니다.
+ 실행에는 kaling.js 과 crypto.js 가 필요합니다.
+ https://hearthstonejson.com/ 에서 카드 json 파일을 불러옵니다.
+ 자동으로 sdcard/퀴즈.txt 에 유저 정보가 저장됩니다.
